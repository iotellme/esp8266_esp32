To install this library, just place this entire folder as a subfolder in your
Arduino/lib/targets/libraries folder.

When installed, this library should look like:

Arduino/lib/targets/libraries/iotellme              (this library's folder)
Arduino/lib/targets/libraries/iotellme/iotellme.h       (the library description file)
Arduino/lib/targets/libraries/iotellme/keywords.txt (the syntax coloring file)
Arduino/lib/targets/libraries/iotellme/examples     (the examples in the "open" menu)
Arduino/lib/targets/libraries/iotellme/readme.txt   (this file)
